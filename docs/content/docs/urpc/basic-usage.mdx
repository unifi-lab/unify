## Server Setup

### 1. Install Dependencies

```package-install
@unilab/urpc-hono @unilab/urpc-client @unilab/urpc-core
```

### 2. Create Server

```typescript
// server.ts
import { URPC } from "@unilab/urpc-hono";
import { WalletPlugin } from "@unilab/uniweb3";
import { Logging } from "@unilab/urpc-core/middleware";

const app = URPC.init({
  plugins: [WalletPlugin],
  middleware: [Logging()],
});

export default {
  port: 3000,
  fetch: app.fetch,
};
```

### 3. Client Usage

```typescript
// client.ts
import { repo, URPC } from "@unilab/urpc-client";
import { WalletEntity } from "@unilab/uniweb3/entities";

URPC.init({
  baseUrl: "http://localhost:3000",
  timeout: 10000,
});

const evmBalanceRes = await repo<WalletEntity>({
  entity: "wallet",
  source: "evm",
}).findOne({
  where: {
    address: "0x...",
    network: "ethereum",
  },
});

const solanaBalanceRes = await repo<WalletEntity>({
  entity: "wallet",
  source: "solana",
}).findOne({
  where: {
    address: "11111111111111111111111111111112",
  },
});
```

## Running the Server

```bash
bun run server.ts
# or
node server.js
```

## Key Features

- 🔌 **Plugin System**: Easy integration with pre-built plugins
- 📊 **Middleware**: Built-in logging and custom middleware support
- 🌐 **Multi-chain**: Support for EVM and Solana networks
- ⚡ **Auto APIs**: REST endpoints generated automatically
- 📝 **Type-safe**: Full TypeScript support
