## Quick Start

Choose the appropriate installation method based on your use case:

### Runs only in a browser or node environment

For client-side applications that need basic entity management functionality:

```package-install
@unilab/urpc @unilab/urpc-core
```

### Hono Server

For server-side applications using the Hono framework:

```package-install
@unilab/urpc-hono @unilab/urpc-core hono @hono/node-server
```

### Next.js

```package-install
@unilab/urpc-next @unilab/urpc-client @unilab/urpc-core
```

### Client Library

For applications that need client-side API calling functionality:

```package-install
@unilab/urpc-client
```

## Optional Packages

### UniWeb3 Plugin

Install when you need Web3 wallet functionality:

```package-install
@unilab/uniweb3
```

### Middleware Support

Install when you need advanced middleware functionality:

```package-install
@unilab/urpc-core
```

## Usage Examples

### Browser Environment

```typescript
import { UserEntity } from "./entities/user";
import { PostEntity } from "./entities/post";
import { URPC, repo } from "@unilab/urpc";
import { WalletPlugin } from "@unilab/uniweb3";
import { createHookMiddleware, Logging } from "@unilab/urpc-core/middleware";

const MyPlugin = {
  entities: [UserEntity, PostEntity],
  adapters: [
    { source: "demo", entity: "UserEntity", adapter: new UserAdapter() },
    { source: "demo", entity: "PostEntity", adapter: new PostAdapter() },
  ],
};

URPC.init({
  plugins: [MyPlugin, WalletPlugin],
  middleware: [Logging()],
});

// Use repo for data operations
const users = await repo({
  entity: "user",
  source: "demo",
}).findMany();
```

### Hono Server

```typescript
import { URPC } from "@unilab/urpc-hono";
import { UserAdapter } from "./adapters/user";
import { PostAdapter } from "./adapters/post";
import { Logging } from "@unilab/urpc-core/middleware";

const MyPlugin = {
  entities: [UserEntity, PostEntity],
  adapters: [
    { source: "demo", entity: "UserEntity", adapter: new UserAdapter() },
    { source: "demo", entity: "PostEntity", adapter: new PostAdapter() },
  ],
};

const app = URPC.init({
  plugins: [MyPlugin],
  middleware: [Logging()],
});

export default {
  port: 3000,
  fetch: app.fetch,
};
```

### Next.js App Router

Create `src/api.ts`:

```typescript
import { URPC } from "@unilab/urpc-next/app-router";
import { WalletPlugin } from "@unilab/uniweb3";
import { Logging } from "@unilab/urpc-core/middleware";

export const api = URPC.init({
  plugins: [WalletPlugin],
  middleware: [Logging()],
});
```

Create `src/app/api/[...URPC]/route.ts`:

```typescript
import { api } from "../../../api";

export const { GET, POST, PATCH, DELETE } = api;
```

### Next.js Pages Router

Create `src/pages/api/[...URPC].ts`:

```typescript
import { URPC } from "@unilab/urpc-next/pages-router";
import { WalletPlugin } from "@unilab/uniweb3";
import { Logging } from "@unilab/urpc-core/middleware";

const handler = URPC.init({
  plugins: [WalletPlugin],
  middleware: [Logging()],
});

export default handler;
```
