## Quick Start with Web3

Unify provides built-in Web3 support for blockchain operations. Here's how to get started with a basic setup:

## Server Setup

Create a `server.ts` file to set up your API server with Web3 adapters:

```typescript
import { Unify } from "@unilab/server";
import { SolanaAdapter, EVMAdapter } from "@unilab/uniweb3";

// Register Web3 adapters
const server = Unify.register([
  { source: "solana", adapter: new SolanaAdapter() },
  { source: "evm", adapter: new EVMAdapter() },
]);

console.log("üöÄ Starting Unify server on port 3000");
console.log("üìä Available endpoints:");
console.log(
  '  GET  /wallet/find_one?source=solana&where={"address":"wallet_address"} - Get Solana wallet balance'
);
console.log(
  '  GET  /wallet/find_one?source=evm&where={"address":"wallet_address","network":"ethereum"} - Get EVM wallet balance'
);

export default {
  port: 3000,
  fetch: server.fetch,
};
```

## Client Usage

Create a `client.ts` file to consume your API:

```typescript
import { repo, UnifyClient } from "@unilab/client";
import { WalletEntity } from "@unilab/uniweb3/entities";

// Initialize the client
UnifyClient.init({
  baseUrl: "http://localhost:3000",
  timeout: 10000,
});

async function demo() {
  try {
    // Query EVM wallet balance
    const evmWallet = await repo<WalletEntity>("wallet", "evm").findOne({
      where: {
        address: "0x4f00D43b5aF0a0aAd62E9075D1bFa86a89CDb9aB",
        network: "iotex",
      },
    });
    console.log("EVM Wallet:", evmWallet);

    // Query Solana wallet balance
    const solanaWallet = await repo<WalletEntity>("wallet", "solana").findOne({
      where: {
        address: "11111111111111111111111111111112",
      },
    });
    console.log("Solana Wallet:", solanaWallet);

  } catch (error) {
    console.error("‚ùå Error:", error.message);
  }
}

demo();
```

## Running the Example

### 1. Start the Server

```bash
# Using Bun
bun run server

# Using Node.js
node server.js
```

### 2. Test with Client

In another terminal:

```bash
# Using Bun
bun run client

# Using Node.js  
node client.js
```

### 3. Manual Testing with curl

You can also test the API manually:

```bash
# Get Solana wallet balance
curl "http://localhost:3000/wallet/find_one?source=solana&where=%7B%22address%22%3A%2211111111111111111111111111111112%22%7D"

# Get EVM wallet balance
curl "http://localhost:3000/wallet/find_one?source=evm&where=%7B%22address%22%3A%220x4f00D43b5aF0a0aAd62E9075D1bFa86a89CDb9aB%22%2C%22network%22%3A%22iotex%22%7D"
```

## API Endpoints

The server automatically generates Web3 API endpoints:

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/wallet/find_one?source=solana` | Get Solana wallet balance |
| GET | `/wallet/find_one?source=evm` | Get EVM wallet balance |

### Query Parameters

#### Solana Wallet Query
```
/wallet/find_one?source=solana&where={"address":"wallet_address"}
```

#### EVM Wallet Query  
```
/wallet/find_one?source=evm&where={"address":"wallet_address","network":"network_name"}
```

**Supported EVM Networks:**
- `ethereum` - Ethereum mainnet
- `polygon` - Polygon
- `bsc` - Binance Smart Chain
- `iotex` - IoTeX
- And more...

### WalletEntity Interface

```typescript
interface WalletEntity extends BaseEntity {
  address: string;
  balance: string;
  network: string;
  token?: {
    symbol: string;
    decimals: number;
  };
}
```
