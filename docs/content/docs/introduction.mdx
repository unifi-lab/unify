# Unify

A Hono-based SDK with built-in Web3 support for unified blockchain data access

## Overview

Unify is a powerful and flexible SDK built on top of the high-performance [Hono](https://hono.dev/) web framework. It provides unified access to multiple blockchain networks through a consistent API, making it easy to build Web3 applications and query blockchain data.

## Features

- 🔧 **Flexible Querying**: Support for complex query parameters and filtering
- 🛡️ **Error Handling**: Built-in error handling and response standardization
- 🔌 **Extensible**: Easy to add custom adapters and middleware
- 💪 **TypeScript**: Full TypeScript support with type safety
- 🔗 **Framework Integration**: Seamless integration with Next.js, Hono and other frameworks
