🔧 使用方法

基础集成

在你的HTML页面中添加以下代码：

<!-- 基础用法 --> <script src="https://studio.uni-labs.org/embed-chat.js"></script> <!-- 或者使用自定义服务器 --> <script src="https://studio.uni-labs.org/embed-chat.js" data-src="https://your-studio-domain.com"></script>

高级配置

<!-- 加载后自定义配置 --> <script> // 设置自定义服务器地址 URPCChat.setStudioUrl('https://your-custom-server.com'); // 程序化打开聊天窗口 URPCChat.open(); // 检查聊天状态 if (URPCChat.isOpen()) { console.log('聊天窗口已打开'); } </script>

🎮 API 方法

加载脚本后，可以使用 window.URPCChat 对象控制聊天功能：

  • URPCChat.open() - 打开聊天窗口
  • URPCChat.close() - 关闭聊天窗口
  • URPCChat.toggle() - 切换聊天窗口状态
  • URPCChat.isOpen() - 检查聊天窗口是否打开
  • URPCChat.setStudioUrl(url) - 设置自定义服务器地址
